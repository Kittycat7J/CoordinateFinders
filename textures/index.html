Block Rotation Detector

How to use:

  1. Upload an image using the button below
  2. Adjust the 4 corner points to define the perspective area (clockwise from top-left)
  3. The transformed image with grid will appear in real-time
  4. Click on any grid square to run detection on it

Controls:

  • Mouse: Click and drag corner points to adjust perspective
  • Number Keys (1-4): Select different corner points
  • Arrow Keys or WASD: Move selected point in small increments
  • +/- Keys: Adjust movement step size
  • R Key: Reset points to default positions
  • Left Click: Run AI detection on grid squares
